Desenvolvimento de um Painel Gerencial para Negócio com Streamlit

Nesse projeto, os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimento web como o Streamlit e Github, foram usados para desenvolver um painel com as principais métricas de uma empresa de marketplace de restaurantes, onde seu principal objetivo é facilitar o encontro e negociações de clientes e restaurantes.

O resultado final do projeto foi um painel hospedado em um ambiente Cloud e disponibilizado através de um link web. O painel pode ser acessado por qualquer dispositivo conectado na internet.

As ferramentas utilizadas foram:

  • Python.

Ensaio de Machine Learning

Nesse projeto, foram explorados os principais conceitos de treinamento, ajuste de parâmetros e limiares entre overfitting e underfitting de vários algoritmos de Machine Learning dentro das tarefas de classificação, regressão e agrupamento (clusterização).

O resultado final do projeto foi um painel mostrando a performance dos algoritmos de Machine Learning, a partir das variações dos valores dos principais parâmetros de ajuste de cada algoritmo para obter a melhor performance nas métricas.

As ferramentas utilizadas foram:

  • Python 3.8 e Scikit-learn
  • Algoritmos de classificação: KNN, Decision Tree, Random Forest e Logistic Regression.
  • Algoritmos de regressão: Linear Regression, Decision Tree Regressor, Random Forest Regressor, Polinomial Regression, Linear Regression Lasso, Linear Regression Ridge, Linear Regression Elastic Net, Polinomial Regression Lasso, Polinomial Regression Ridge e Polinomial Regression Elastic Net.
  • Algoritmos de agrupamento: K-Means e Affinity Propagation.
  • Métricas de performance: Accuracy, Precision, Recall, F1-Score, R2, MSE, RMSE, MAE, MAPE e Silhouette Score.

Previsão de vendas das lojas do Grupo Rossmann

Nesse projeto, nos deparamos com uma motivação do CFO em saber as previsões de vendas de todas as lojas para as próximas 6 semanas. Tendo em vista que as lojas realizavam a previsão de vendas de forma manual e com base em experiências passada, o time de Ciência de Dados optou no uso de Machine Learning para realizar a previsão das vendas de todas as lojas.

Um outro obstáculo observado pelo time de Ciência de Dados, foi a visualização das previsões sendo limitada apenas ao computador. Sabendo disso, o time de Ciência de Dados desenvolveu uma forma de receber as informações pelo Smartphone.

O resultado final do projeto foi um BOT, via Telegram, onde o usuário envia o código ID da loja que ele deseja saber as previsões de vendas e com isso o BOT retorna o valor de faturamento previsto para as próximas 6 semanas.

As ferramentas utilizadas foram:

  • Python
  • Algoritmos de Machine Learning
  • Jupyter Lab.
  • Webservice do Render
  • Github.
